🆘 TROUBLESHOOTING QUICK LINKS
================================================================================

Problem                      Solution
─────────────────────────────────────────────────────────────────
Cannot connect to MongoDB  → Check MONGODB_URI, IP whitelist
API returns 401            → Check JWT token, JWT_SECRET
Razorpay payment fails     → Use test keys, test card
Frontend can't reach API   → Check CORS config, API_BASE_URL
Build errors               → Run pnpm install, clear node_modules
Admin login not working    → Check credentials, JWT_SECRET
Images not uploading       → Check Multer config, file size
Mobile layout broken       → Check responsive.css breakpoints
